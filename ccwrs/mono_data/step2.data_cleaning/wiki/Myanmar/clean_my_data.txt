This pipeline:
cleans text
removes noise
splits into sentences
filters invalid sentences
removes duplicates
randomly samples 20k sentences
outputs final clean corpus for mining/pretraining

==============

✅ Cleans text — strips URLs, HTML tags, empty brackets, non-Burmese/OCR garbage characters, normalizes repeated punctuation/whitespace
✅ Removes noise — the OCR garbage regex and template-heavy census-sentence filter (is_template_heavy)
✅ Splits into sentences — on ၊။!? (Burmese-aware segmentation)
✅ Filters invalid sentences — token count bounds (4–120), too much English, too many symbols, numeric-only lines, repeated-character runs
✅ Removes duplicates — via a normalized (lowercased, punctuation-stripped) seen set
✅ Randomly samples — shuffles with a fixed seed (42) then truncates
✅ Outputs a final corpus
